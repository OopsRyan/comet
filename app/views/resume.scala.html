Super Learner

Stacked Ensemble Algorithm ☞ Demo

The Super Learner is a heterogeneous stacked ensemble classifier described in Super Learner In Prediction(van der Laan et al, 2007). This is a classification model that uses a set of base classifiers of different types, the output of which are combined in another classifier at the stacked layer.

The figure below shows a flow diagram of the Super Learner process. The base classifiers are trained and their outputs are combined along with the training dataset labels into a training set for the stack layer classifier. To avoid overfitting the generation of the stacked layer training set uses a k-fold cross validation process (described as V-fold in the figure).

The super learner class inheriting from the base estimator class of scikit-learn is compatible with its APIs like GridSearchCV. The classifier at the stacked layer can be trained on label or probability outputs from the base classifiers by indicating a parameter. Also provided a method to measure the correlation between base classifiers for improving prediction performance.
